# Daily Digest — личный командный центр

Самостоятельный сервис (без Claude), который каждое утро собирает в одну сводку:

- 📬 **Почту** (Gmail по IMAP) — выделяет письма, требующие действия
- 📰 **Новости** (любые RSS-ленты)
- 🌤 **Погоду** (Open-Meteo, по умолчанию Москва)
- 💱 **Курсы валют** (ЦБ РФ: USD, EUR к рублю)
- 📌 **Список дел** — автоматически из писем
- 💡 **Рекомендации** — на основе текущей картины

И сверху — **умный LLM-слой** (опционально), который делает сводку персональной:

- 🌅 **Утреннее обращение** — личное, на «ты», с учётом твоего профиля и картины дня
- 🧠 **AI-приоритеты дел** — сортировка задач по важности с пояснением
- ✉️ **AI-разбор почты** — приоритет и конкретное действие по каждому письму
- 📰 **AI-выжимка новостей** — суть и тренды вместо списка заголовков

Поддерживаются 4 провайдера: **OpenAI, Anthropic (Claude), Grok (xAI), Gemini** —
переключаются одной строкой в `.env`. Без ключа сервис работает как обычно
(умные блоки просто не показываются).

Доставка: **Telegram-бот** + **веб-страница** (дашборд, всегда свежий).
Все источники погоды/курсов/новостей работают **без API-ключей**.

🔐 **Вход на дашборд — по QR-коду через Telegram**: открываешь страницу,
сканируешь QR телефоном, подтверждаешь в боте — и попадаешь внутрь.
Без паролей. Доступ только владельцу (твой `CHAT_ID`).

📦 **Деплой на свой VPS** с доменом и автоматическим HTTPS — см.
[DEPLOY.md](DEPLOY.md).

## Быстрый старт (Docker, рекомендуется)

### 1. Подготовьте доступы

**Gmail (IMAP):**
1. Включите двухфакторную аутентификацию в Google-аккаунте.
2. Создайте App Password: https://myaccount.google.com/apppasswords
   (выберите «Почта» / «Другое»). Получите 16-значный код.

**Telegram:**
1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot` → получите **токен**.
2. Напишите своему боту любое сообщение.
3. Откройте `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates` —
   в ответе найдите `"chat":{"id": ...}`. Это ваш **CHAT_ID**.

### 2. Настройте окружение

```bash
cp .env.example .env
nano .env   # впишите GMAIL_*, TELEGRAM_* и при желании поменяйте город/валюты
```

### 3. Запустите

```bash
docker compose up -d --build
```

Готово:
- Веб-дашборд: `http://<IP-вашего-сервера>:8080`
- Сводка в Telegram приходит ежедневно в `DIGEST_TIME` (по умолчанию 08:00 МСК)

Проверить вручную (отправит сводку в Telegram прямо сейчас):

```bash
docker compose run --rm digest python main.py run
```

Логи:

```bash
docker compose logs -f
```

## Запуск без Docker

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # заполнить
python main.py serve   # веб + планировщик
# или разовый прогон:
python main.py run
```

## Режимы запуска

| Команда | Что делает |
|---|---|
| `python main.py serve` | Веб-сервер + встроенный планировщик (основной режим) |
| `python main.py run`   | Разовый сбор и отправка в Telegram (для cron или теста) |
| `python main.py web`   | Только веб-сервер, без планировщика |

### Альтернатива: системный cron вместо встроенного планировщика

Если хотите управлять расписанием через cron хоста, запускайте контейнер
в режиме `web` (только дашборд), а отправку вешайте на cron:

```cron
0 8 * * * cd /path/to/daily-digest && docker compose run --rm digest python main.py run
```

## Настройка под себя (.env)

| Переменная | Назначение |
|---|---|
| `GMAIL_MAX_EMAILS` | Сколько последних писем анализировать |
| `GMAIL_UNREAD_ONLY` | `true` — только непрочитанные |
| `NEWS_FEEDS` | RSS-ленты через запятую — добавляйте свои источники |
| `NEWS_MAX_ITEMS` | Сколько новостей показывать |
| `CURRENCIES` | Коды валют к рублю (напр. `USD,EUR,CNY`) |
| `WEATHER_LAT/LON/CITY` | Координаты и название города |
| `DIGEST_TIME` | Время утренней сводки, `HH:MM` |

### Умный LLM-слой

| Переменная | Назначение |
|---|---|
| `LLM_PROVIDER` | `openai` / `anthropic` / `grok` / `gemini`. Пусто = выключить |
| `LLM_MODEL` | Конкретная модель (пусто = дефолт провайдера) |
| `OPENAI_API_KEY` / `ANTHROPIC_API_KEY` / `GROK_API_KEY` / `GEMINI_API_KEY` | Нужен только тот, что соответствует провайдеру |
| `PROFILE_NAME` | Твоё имя (для обращения) |
| `PROFILE_ROLE` | Чем занимаешься — ассистент учитывает в приоритетах |
| `PROFILE_PRIORITIES` | Текущие приоритеты — влияет на сортировку дел и почты |
| `PROFILE_INTERESTS` | Интересы — влияет на выжимку новостей |
| `PROFILE_EXTRA` | Любой доп. контекст о тебе |

Чем подробнее профиль — тем точнее ассистент «попадает в тебя».
Где взять ключи: OpenAI — platform.openai.com, Anthropic — console.anthropic.com,
Grok — console.x.ai, Gemini — aistudio.google.com.

«Дела на сегодня» формируются из писем, тема которых содержит сигнальные слова
(«срочно», «дедлайн», «счёт», «invoice», «требуется» и т.д.).
Список слов можно расширить в `app/sources/mail.py` → `ACTION_KEYWORDS`.

## Развитие

- **Telegram как источник** (чтение каналов/чатов): официального IMAP-аналога нет.
  Можно добавить чтение публичных каналов через Telegram Bot API
  (бот должен быть админом канала) или через Telethon (user-сессия) — отдельный модуль в `app/sources/`.
- **Другая почта**: модуль `mail.py` работает с любым IMAP — поменяйте `IMAP_HOST`.
- **Умные рекомендации/саммари новостей**: подключить LLM-API в `app/digest.py`.

## Безопасность

`.env` содержит пароли и токены — он в `.gitignore` и `.dockerignore`.
Не коммитьте его и не открывайте порт 8080 в публичный интернет без
reverse-proxy с авторизацией (дашборд показывает вашу почту).
